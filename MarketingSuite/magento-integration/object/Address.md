# Address object

Address object represents a specific location in the world. Inside Magento there are addresses assigned to customer, quotes and orders. 

Properties:

- _number_ **ID**
- _[Customer](magento/documentation/Customer.md)_ **customer**
- _string_ **street**
- _string_ **city**
- _string_ **zipcode**
- _string_ **state**
- _string_ **country**
- _string_ **phone**
- _string_ **fax**
- _string_ **company**
